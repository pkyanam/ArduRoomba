Library Features

Core Functionality

Complete implementation of iRobot Open Interface (OI) specification with movement commands, sensor data reading, and cleaning mode controls.

  • Movement control (forward, backward, turn, drive)
  • Sensor data reading (battery, bumpers, cliff sensors)
  • Cleaning modes (clean, spot, dock)
  • Safe and full control modes
  • Comprehensive error handling

  • WiFi connection management
  • REST API endpoints
  • Web-based control interface
  • Real-time sensor monitoring
  • Cross-Origin Resource Sharing (CORS)

Getting Started

Follow these steps to set up ArduRoomba with your hardware

1

Install the Library

Download and install the ArduRoomba library in your Arduino IDE

// Option 1: Arduino Library Manager // Search for "ArduRoomba" and click Install // Option 2: Manual Installation // Download ZIP from GitHub and add to Arduino IDE // Sketch ‚Üí Include Library ‚Üí Add .ZIP Library
2

Wire Your Hardware

Connect your Arduino to the Roomba's Open Interface connector

Arduino Pin 2 ‚Üí Roomba TX
Arduino Pin 3 ‚Üí Roomba RX
Arduino Pin 4 ‚Üí Roomba BRC
Arduino GND ‚Üí Roomba GND
Arduino VCC ‚Üí Roomba VCC

Note: Use appropriate voltage levels and consider using a voltage divider for 5V boards.

3

Basic Usage

Start with a simple example to test your connection

#include "ArduRoomba.h" // Create ArduRoomba instance ArduRoomba roomba(2, 3, 4); // rxPin, txPin, brcPin void setup() { Serial.begin(19200); // Initialize Roomba ArduRoomba::ErrorCode result = roomba.initialize(); if (result == ArduRoomba::ErrorCode::SUCCESS) { Serial.println("Roomba connected!"); roomba.safe(); // Enter safe mode } } void loop() { // Move forward for 2 seconds roomba.drive(200, 0); delay(2000); // Stop roomba.drive(0, 0); delay(1000); }
4

WiFi Control (Optional)

For WiFi-enabled boards, add wireless control capabilities

#include "ArduRoomba.h" #include "wireless/ArduRoombaWiFi.h" ArduRoomba roomba(2, 3, 4); ArduRoombaWiFi wifi(roomba); void setup() { Serial.begin(115200); // Initialize Roomba roomba.initialize(); // Connect to WiFi wifi.begin("YourWiFiNetwork", "YourPassword"); // Start web server wifi.startWebServer(80); Serial.println("Access control at: " + wifi.getServerURL()); } void loop() { wifi.handleClient(); }
