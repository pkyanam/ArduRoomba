Get Started in Minutes

Follow these simple steps to transform your Roomba into a smart wireless robot

1

Install Library

Install ArduRoomba through Arduino IDE Library Manager or download from GitHub

// Arduino IDE: Sketch ‚Üí Include Library ‚Üí Manage Libraries
// Search for "ArduRoomba" and click Install
2

Wire Your Hardware

Connect your Arduino board to the Roomba using the provided wiring diagram

Pin 2 ‚Üí Roomba TX
Pin 3 ‚Üí Roomba RX
Pin 4 ‚Üí Roomba BRC
3

Upload Code

Use one of our examples or create your own sketch with just a few lines of code

#include "ArduRoomba.h"
#include "wireless/ArduRoombaWiFi.h"

ArduRoomba roomba(2, 3, 4);
ArduRoombaWiFi wifi(roomba);

void setup() {
  roomba.initialize();
  wifi.begin("YourWiFi", "password");
  wifi.startWebServer();
}
4

Control Wirelessly

Access the web interface from any device and start controlling your Roomba

http://192.168.1.100
